1. Getting Started

This chapter describes the basic usage of the cfeclipse plug-in. In this help, we assume you have a familiarity with the eclipse IDE (Integrated Development Environment). If you have never used the eclipse IDE, you might benefit from a quick read of the overview of the basic IDE usage; however, one should still be able to follow this help without knowing the ins and outs of the IDE.

1.1. Creating A Cold Fusion Project

The Cold Fusion editor (CFML Editor) will only activate properly if within a project. Trying to simply open a cold fusion file will cause the editor to fail. This is an error in the plug-in and slated to be fixed To create a Cold Fusion project select File then New followed by Project...; these steps will launch the new project wizard shown below.

Figure 1. New Project Wizard

In the new wizard dialog, select Simple then Project, and then click next.

On the next step you will need to give a name to the project, and specify the directory where the local copy of the files should be stored (see diagram below). If you have cold fusion files that already exist on your system, you can specify the directory to use by selecting the Browse... button and choosing the appropriate directory.

Note

Selecting a directory other then the default will not remove any existing files it will simply use that directory as the project directory and create a few project specific files.

Figure 2. New Project Wizard Step 2

After the project name and directory have been specified, click finish to create the new project. This will add a project folder to your navigation view on your workbench where you can create, edit, and delete cold fusion files.

1.2. Adding and editing Cold Fusion Files

The CFML Editor automatically assigns it's self to *.cfm, *.cfml, and *.cfc files so when any of these types of files are created or edited, the CFML Editor will automatically start. This holds true even if you have a Java™ or other type of project defined and want to use cold fusion pages within those projects.

Note

Other plug-ins can take over the cold fusion file extension associations. If you have another plug-in installed that handles cold fusion files, or you install a plug-in after cfeclipse that can edit cold fusion files, you may need to reassign the cold fusion file extensions back to the CFML Editor.

To create a cold fusion file simply select File, New, then File. After the new file wizard begins (see image below), select the project and folder you wish to create the file in, enter the file name and give it a proper cold fusion extension, and then click finish. Eclipse will create the file, and open the new file in the CFML Editor.

Figure 3. New File Wizard

To edit a cold fusion file simple double click the file name in the navigation view on the main workbench view.
