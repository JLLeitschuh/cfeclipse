Release Notes - 1.1.13

7th July, 2004 : 19:22

  • CFML syntax dictionary has some minor improvements
  • Improved auto-insertion, tag closing & indenting
  • Preference page for code auto-insertion, tag closing & indenting. You can find it under "CFEclipse->Auto-insertion & closing"
  • Implemented feature issue #67: "Right-click <cfinclude> tag "open included file" in context menu"
  • Implemented not only for <cfinclude> but also for <cfmodule>
  • Implemented quote step through on manually inserted tag attributes
  • Added File Parser preference page. You can turn off Problem reporting (clears all current problems as well)
  • Return types shown in CFC methods view
