Announcement: CFEclipse version 1.1.15 "Sonic" released.

22nd July, 2004 : 22:08

Installation & update procedures are listed at the bottom of this mail. All current non-CVS users of CFEclipse should hopefully be notified by Eclipse that there is a new release available. If not, follow the update procedures and mention it to us on the CFE lists.

The Scribble Pad:

As of 1.1.15 CFE comes with a major new feature: the 'scribble pad'. The Scribble Pad is intended to allow you to test those random small ideas.

To use the Scribble Pad you must first setup a project named 'Scribble' and create a file called 'scribble.cfm'. scribble.cfm must be a file that is on your web server root. Now find the CFEclipse preference pages and change the 'URL to scribble file' in the Scribble Pad node to wherever the scribble.cfm file resides on your web server.

Now from any project you can hit F8 and your scribble file will be opened. Stick some code in, hit F8 again and the browser will display the result of that file.

The Scribble Pad is very much a work-in-progress, so any and all ideas on it are welcome. Best place is the CFE mailing lists at http://cfeclipse.tigris.org As a hint, current ideas are indexing of saved scribbles, keyword association and saved-scribble-searches to find that piece of test code that you've lost.

Other features you may enjoy:

  • Snippet triggers: You can now assign trigger text to each snippet. Type it in, hit Ctrl+I and your snippet is inserted.
  • Loopy snippets: Enter your snippet trigger text plus a "*2" and get double the snippet fun.
  • CFC Method view now has the method hints shown
  • CFC Method view has an alphabetic sorting button
  • Double clicking a CFC method name now sends focus to the editor.
  • Context sensitive CFML F1 help: takes your browser to the appropriate livedocs page, otherwise to Google you go.
  • No more duplicate attribute suggestions: type '
  • 'Jump to end tag' context menu implemented.
  • Syntax colouring is now working. Window->Preferences->CFEclipse and then either CFML Colors or HTML Colors.
  • '=' no longer causes content/code assist to disappear

Bugs fixed:

  • #124 "code hints suddenly stop working"
  • #101 "Need to check something in the content tree before usage"
  • #156 "Paste attribute causes problem at end of tag"
  • #148 "keyword coloring a bit off on tags"

Known issues:

  • 'Jump to end tag' is only reliable after post-edit saving.
  • Changes to syntax colours are not reflected in currently open editor windows. They must be closed and re-opened.

A zip version is available on the CFEclipse website if you wish to perform the installation/update manually.
