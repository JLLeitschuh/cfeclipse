CFEclipse is a CFML (Cold Fusion Markup Language) plug-in for Eclipse. This plug-in adds a CFML script editor, syntax highlighting, and code insight for the CFML language. It also provides code insight for HTML 4.01, a specialized cfscript editor, and a snippet (HTML fragment) inserter. CFML is a server side scripting language that sits on top of a J2EE server. There are currently two companies that make engines that can process CFML scripts - Macromedia and New Atlanta.

Mission

This project is attempting to create the best environment for writing CFML code. This project does not intend to create an HTML WYSIWYG editor, or an editor with large numbers of wizards. This project is attempting to make a CFML editor for coders.

Installation

To install the plug-in do the following:

  1. Download the Eclipse IDE version 3.0 or higher
  2. download a plug-in zip file
  3. Unzip the file into your eclipse/plugin directory
  4. restart Eclipse
  5. If you have an editor already linked to cfm, cfc, cfml, htm, or html files you'll need to change the editor setting in Window/Preferences then in the tree select Workbench/File Associations, and set cfm, cfc, cfml, htm and html to the CFML Editor.
