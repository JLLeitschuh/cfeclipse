This URL is where the cfeclipse dictionary namespace is mapped.

The dictionary syntax file can be found here as well

To use the dictionary, the XML file should start with the following:

<dictionary xmlns="http://www.cfeclipse.org/version1/dictionary"
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://www.cfeclipse.org/version1/dictionary http://www.cfeclipse.org/version1/dictionary/dictionary.xsd"
>
...
</dictionary>
