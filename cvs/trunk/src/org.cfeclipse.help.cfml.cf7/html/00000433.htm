CreateODBCTime

Description

Creates an ODBC time object.

Returns

A time object, in ODBC timestamp format.

Category

Date and time functions

Function syntax

CreateODBCTime(date)

See also

CreateODBCDateTime, CreateTime, Evaluation and type conversion issues in Using ColdFusion Variables in ColdFusion MX Developer's Guide

Parameters

Parameter Description

date

Date/time object in the range 100 AD-9999 AD.

Usage

When passing a date-time value as a string, you must enclose it in quotation marks. Otherwise, it is interpreted as a number representation of a date-time object.
