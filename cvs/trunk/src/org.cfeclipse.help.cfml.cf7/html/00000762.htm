getNickName

Description

Returns the gateway's nickname (display name), if it has been set using the gatewayHelper object setNickName method.

Syntax

string = getNickName()

See also

getName, getProtocolName, numberOfMessagesReceived, numberOfMessagesSent, setNickName, Using the GatewayHelper object in Using the Instant Messaging Event Gateways of ColdFusion MX Developer's Guide

Returns

The gateway's nickname, if any; empty string, otherwise.

Example

See GatewayHelper example, in Using the Instant Messaging Event Gateways in ColdFusion MX Developer's Guide, which uses all GatewayHelper class methods.
