Building and Using ColdFusion Components

This chapter describes Macromedia ColdFusion component (CFC) concepts and elements and tells you how to create and use CFCs. It provides you with all the tools to create and use CFCs; however, it does not discuss CFC methodologies or provide information on designing applications for CFC use. For more information on creating applications that use CFCs, see the Macromedia website: www.macromedia.com.
