getDenyList

Description

Returns the list of users that the IM server has been told not to send state information about the gateway, if the permit mode is set to DENY_SOME.

Syntax

array = getDenyList()

See also

addDeny, addPermit, getPermitList, getPermitMode, removeDeny, removePermit, setPermitMode, Using the GatewayHelper object in Using the Instant Messaging Event Gateways of ColdFusion MX Developer's Guide

Returns

An array of IDs (buddy names) of the users on the gateway's deny list, the list of IDs to which the IM server does not send presence status information.

Note: XMPP permission management is included in the XMPP 1.0 draft specification, but several XMPP servers that were available at the time of the ColdFusion MX 7 release do not support permission management. If the server does not support permission management, this function always returns False.

Example

See GatewayHelper example, in Using the Instant Messaging Event Gateways in ColdFusion MX Developer's Guide, which uses all GatewayHelper class methods.
