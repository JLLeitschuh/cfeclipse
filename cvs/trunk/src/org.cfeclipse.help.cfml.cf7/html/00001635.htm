The eventgateway.log file

Event gateways provided with ColdFusion MX log event gateway errors and events to the cf_root\WEB-INF\cfusion\logs\eventgateway.log file on J2EE configurations, or the cf_root\logs\eventgateway.log file on server configurations. ColdFusion MX includes methods that let any event gateway use this file. This log file can be very useful in debugging event gateways and event gateway applications.
