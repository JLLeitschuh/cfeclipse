getBuddyList

Description

Gets the buddy list for the gateway's user ID.

Syntax

array = getBuddyList()

See also

addBuddy, getBuddyInfo, removeBuddy, Using the GatewayHelper object in Using the Instant Messaging Event Gateways in ColdFusion MX Developer's Guide

Returns

An array of IDs (buddy names) of the users on the gateway's buddy list, a list of instant messaging IDs that this gateway normally communicates with.

Example

See GatewayHelper example, in Using the Instant Messaging Event Gateways in ColdFusion MX Developer's Guide, which uses all GatewayHelper class methods.
