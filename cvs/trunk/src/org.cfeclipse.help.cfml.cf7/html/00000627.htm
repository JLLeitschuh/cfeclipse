Sgn

Description

Determines the sign of a number.

Returns

  • 1, if number is positive.
  • 0, if number is 0.
  • -1, if number is negative.

Category

Mathematical functions

Function syntax

Sgn(number)

See also

Abs

Parameters

Parameter Description

number

A number
