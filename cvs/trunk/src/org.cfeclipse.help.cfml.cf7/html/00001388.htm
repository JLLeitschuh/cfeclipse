Creating Forms in Macromedia Flash

This chapter describes how to create effective forms in Macromedia Flash format. In this format, Macromedia ColdFusion MX 7 displays forms using Flash, not HTML. This chapter describes Flash forms and how they differ from HTML forms, and discusses how to create Flash forms.
