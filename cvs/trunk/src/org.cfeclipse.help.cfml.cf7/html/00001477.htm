Using the Flash Remoting Service

Using the Macromedia Flash Remoting service of Macromedia ColdFusion MX 7, ColdFusion MX developers can work together with Macromedia Flash MX 2004 designers to build dynamic Flash user interfaces for ColdFusion MX applications.

This chapter describes how ColdFusion MX interacts with Flash Remoting for Flash MX 2004 ActionScript 2.0. For a complete description of Flash Remoting capabilities, including how ColdFusion MX interacts with Flash Remoting, see Using Flash Remoting MX 2004 and Flash Remoting ActionScript Dictionary in Flash Help. You can also access the Flash Remoting documentation on the Flash Remoting Developer Center at www.macromedia.com/devnet/mx/flashremoting.
