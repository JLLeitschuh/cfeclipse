cfsilent

Description

Suppresses output produced by CFML within a tag's scope.

Category

Data output tags, Page processing tags

Syntax

<cfsilent>
...
</cfsilent>

See also

cfcache, cfflush, cfheader, cfhtmlhead, cfinclude, cfsetting; Writing and Calling User-Defined Functions in ColdFusion MX Developer's Guide

Usage

This tag requires an end tag.

Example

<h3>cfsilent</h3>

<cfsilent>
  <cfset a = 100>
  <cfset b = 99>
  <cfset c = b-a>
  <cfoutput>Inside cfsilent block<br>
  b-a = #c#</cfoutput><br>
</cfsilent>

<p>Even information within cfoutput tags does not display within a 
cfsilent block.<br>
<cfoutput>
    b-a = #c#
</cfoutput>
</p>
