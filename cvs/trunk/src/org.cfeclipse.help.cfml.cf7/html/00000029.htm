Installing the J2EE Configuration

This chapter describes how to install, deploy, and undeploy Macromedia ColdFusion MX 7 Enterprise Edition in the J2EE configuration. It includes overview information, specific instructions for installing Macromedia JRun 4 (which automatically deploys and configures ColdFusion MX 7), and general instructions for use with all J2EE application servers.
