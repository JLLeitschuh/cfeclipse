Log10

Description

Calculates the logarithm of number, to base 10.

Returns

Number; the logarithm of number, to base 10.

Category

Mathematical functions

Function syntax

Log10(number)

See also

Exp, Log

Parameters

Parameter Description

number

Positive real number for which to calculate the logarithm

Example

<h3>Log10 Example</h3>
