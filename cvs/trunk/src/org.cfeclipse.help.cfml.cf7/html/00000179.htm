Lesson 10: Restricting Access to ColdFusion Applications

This chapter provides information on using the Login Wizard to restrict access to a new or existing Macromedia ColdFusion application. You will learn about the Login Wizard, and then add user authentication to the Compass Travel application.

This lesson contains the following exercises:

Exercise 1: About the Login Wizard.
Exercise 2: Requiring users to log in to the Compass Travel application.
